# Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio Global Solution FIAP 2025 - O Futuro do Trabalho, aplicando conceitos de Programação Dinâmica para resolver um problema real de otimização de recursos na requalificação profissional.

A solução utiliza o Problema da Mochila para maximizar o valor das habilidades adquiridas por profissionais em transição de carreira, considerando restrições de tempo e recursos disponíveis.

# Contexto do Desafio

## Com base no material fornecido sobre o futuro do trabalho, **onde:**
- 25% das profissões devem mudar radicalmente até 2027 (Fórum Econômico Mundial)
- 65% dos trabalhadores precisam se requalificar para manter empregabilidade
- 170 milhões de novos empregos serão criados entre 2025-2030

## 1. Formulação do Problema

- **Entrada:** 20 habilidades com tempo necessário e valor de mercado
- **Restrição:** Capacidade máxima de horas disponíveis
- **Objetivo:** Maximizar o valor total das habilidades adquiridas
